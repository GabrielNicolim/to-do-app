# To Do App

A simple to do app using React

# Intalation

To run this project you need to install the dependencies using the following commands:

## Api

```
    cd api
```

```
    npm install
```

## Client

```
    cd client
```

```
    npm install
```

# Running The Project

To start the project you need to run the following commands:

## Api

```
    npm start
```

## Client

```
    npm run dev
```
